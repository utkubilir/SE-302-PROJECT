Exam Timetable Planner

Exam Timetable Planner is a standalone Windows desktop application that automatically generates optimized exam schedules for educational institutions. The system processes courses, classrooms, student enrollments, and predefined business constraints to produce a conflict-free and efficient exam timetable.

Target Users

The primary users of this application are Student Affairs personnel responsible for creating and maintaining examination schedules.

Key Features

  • Import course, classroom, and enrollment data from CSV files
  • Automatically generate an optimized exam timetable
  • View the master exam schedule in a structured table format
  • Search schedules by student ID or course code
  • Manually modify exam times and classrooms
  • Instant validation of all manual changes against scheduling constraints
  • Export the final timetable for external use

Scheduling Constraints

  • Exams are scheduled between 09:00 and 18:30
  • A student cannot have more than two exams on the same day
  • A minimum 3-hour gap is enforced between exams on the same day
  • A classroom cannot be assigned to multiple exams at the same time
  • Classroom capacity must be sufficient for the number of students

System Architecture

The application follows a layered architecture to ensure modularity, maintainability, and separation of concerns.

  • Domain Layer – Core data models such as Course, Student, Classroom, and Exam
  • Services Layer – Scheduling, constraint checking, and optimization logic
  • Persistence Layer – CSV data loading and SQLite database management
  • Presentation Layer – JavaFX user interface and controllers

How to Run the Project

  1. Clone the GitHub repository
  2. Open the project in a Java IDE (IntelliJ IDEA or Eclipse)
  3. Ensure Java and JavaFX are properly configured
  4. Run the MainApp class
  5. Load input CSV files and start timetable generation
